# 5. Tính năng em thấy ấn tượng nhất

Theo em, tính năng ấn tượng nhất là **Booking Search và Booking Management** của dự án HBS.

Đây là hệ thống booking khách sạn theo mô hình multi-tenant, tích hợp thanh toán GMO Payment.

Tính năng cho phép:

* tìm kiếm phòng
* tính giá theo nhiều điều kiện
* kiểm tra tình trạng phòng
* quản lý booking
* cập nhật trạng thái thanh toán
* đồng bộ dữ liệu giữa nhiều hệ thống

### Khó khăn

Khó khăn lớn nhất là nghiệp vụ booking rất phức tạp.

Ví dụ:

* Giá phòng thay đổi theo từng ngày.
* Có nhiều chương trình khuyến mãi.
* Mỗi khách sạn có chính sách khác nhau.
* Booking phải đồng bộ trạng thái giữa nhiều màn hình.
* Một thay đổi nhỏ có thể ảnh hưởng tới nhiều chức năng khác.

Ngoài nghiệp vụ, lượng dữ liệu trả về từ API khá lớn nên cần tối ưu render và giảm số lượng request.

Em đã chia nhỏ logic thành các composable, tối ưu việc gọi API, lazy loading và cache dữ liệu để cải thiện hiệu năng.

Thông qua dự án này em học được rất nhiều về cách phân tích nghiệp vụ cũng như thiết kế code có khả năng mở rộng.
